# 🖌️ Virtual White Board

## 🖥️ Features

- ✏️ **Freehand Drawing** – Draw anything using your mouse like on a real whiteboard
- ⚡ **Real-time Rendering** – Instant visual feedback as you move your mouse
- 🧼 **Clean & Minimal UI** – Focused design with no clutter
- 💻 **Built with Pure Java** – Uses only core Java and Swing (no external libraries)
- 🖱️ **Mouse Event Handling** – Implements `MouseListener` and `MouseMotionListener`
- 🧠 **Educational Project** – Great for learning Java GUI, graphics, and event handling
- 🎯 **Customizable** – Easy to extend with colors, brush size, or saving feature

## ⚙️ Requirements

To run this project on your system, make sure you have:

- ✅ **Java Development Kit (JDK)** – Version **8 or above**
- ✅ A **Java-compatible IDE** (like IntelliJ, Eclipse, or NetBeans) OR just the **command line**
- ✅ Basic understanding of Java and GUI concepts (optional, but helpful)

You can check your Java version by running:

```bash
java -version
